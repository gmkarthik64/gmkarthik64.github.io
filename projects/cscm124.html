CSCM124 Final Project

Project Info

I worked on a haplotype assembly project which involved sequencing haplotypes from reads. The problem is as follows:
To sequence haplotypes, reads are made from regions of haplotypes and combined to reassemble the haplotype. The issue is that humans have 2 haplotypes per chromosome (and plants have more), but each read comes from only one haplotype and we do not know which one. The goal is to determine which haplotype each read came from, so they can be correctly reassembled.
For my project, I decided to deal with the case of up to 4 haplotypes and combining the reads while also seeing how increasing the lengths of reads affected the accuracy in determining the original haplotypes.

Project Goal

By the end of the quarter, I wanted to create 3 algorithms: a brute force, greedy, and a more optimized algorithm that would be more accurate than the greedy one, though likely slower. I achieved this by adding a hill-climbing search to my algorithm in order to find solutions similar to the one the greedy alogrithm returned but more fitting of the haplotype reads.
